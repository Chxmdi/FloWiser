# Phase 1 Scope

## Goal
Deliver the smallest credible FloWiser product foundation that can ingest, store, and display site visibility safely.

## Included
- monorepo workspace and tooling
- local development foundation
- backend service shell
- frontend shell
- infrastructure bootstrap templates
- engineering standards
- CI/CD, migrations, and release hooks

## Explicitly not included
- full ingestion logic
- rules engine
- optimization engine
- live customer auth integration
- production-ready infrastructure tuning
